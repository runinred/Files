# Анализ бизнес-показателей развлекательного приложения Procrastinate Pro+

[Ссылка на проект](https://github.com/runinred/Files/blob/main/Mobile%20app%20business%20performance%20analysis/Mobile%20app%20business%20performance%20analysis.ipynb)

## Описание проекта

Несмотря на огромные вложения в рекламу, последние несколько месяцев компания терпит убытки. Требуется разобраться в причинах и помочь компании выйти в плюс.



## Навыки и инструменты

- **python**
- **pandas**
- **matplotlib**
- **numpy**
- **когортный анализ**
- **юнит-экономика**
- **продуктовые метрики**




## Вывод

Проведен анализ данных. Рассчитаны различные метрики, использован когортный анализ: LTV, CAC, Retention rate, DAU, WAU, MAU и т.д. Использованы уже написанные ранее функции расчёта метрик. Исследованы причины неокупаемости рекламы. Сделаны выводы и даны рекомендации.
